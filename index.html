Rules

Guidelines

Eligibility

  • Open to all college students across India
  • Team size must be 3 to 6 members
  • All team members must be from the same college
  • Each participant can be part of only one team
  • Valid college ID required for verification

Submission Rules

  • Projects must be original work created during the event timeline
  • Working prototype is mandatory for submission
  • Code must be hosted on GitHub (public repository)
  • Project report and demo video are required
  • Use of AI tools is permitted but must be disclosed
